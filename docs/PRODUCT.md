# Product

## Definition

ResumeEvolver is a private-first career evidence ledger for one GitHub-heavy technical operator or side-project builder.

The product captures reusable work evidence first, then uses that evidence to draft traceable resume bullets and changelog entries.

## Core value

- trustworthy career evidence
- monthly usefulness over one-time polish
- reusable proof, not polished fiction

## Inputs in v1

- manual evidence entry
- GitHub candidate evidence import
- certification entry
- project link entry
- milestone entry

## Outputs in v1

- role-targeted resume bullet drafts
- monthly or quarterly changelog drafts
- markdown export
- plain text export
- JSON backup

## Out of scope

- LinkedIn integration
- public profile pages
- social posting
- PDF export
- artifact uploads
- commit clustering
- non-GitHub connectors
- team features
- ATS features
